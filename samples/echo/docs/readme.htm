Echo Sample

Overview

This sample is a SIP application. It demonstrates how SIP servlet works.

The SIP servlet - EchoServlet - demonstrates how a simple user agent server handles SIP INVITE and MESSAGE messages. The servlet echo each chat message the client sends.The following diagram shows the typical call flow between X-Lite and EchoServlet.


Understanding the Sample

EchoServlet handles SIP INVITE, BYE, MESSAGE and INFO messages.

EchoServlet always responds positively for all INVITE and BYE messages. The session is implicitly managed by the Prism Application Server. EchoServlet also demonstrates how to manipulate Session Description Protocol (SDP) in the INVITE message.

When the EchoServlet receives SIP MESSAGE message, it confirms the acceptance with 200 OK response. Then it echoes back the message.

For more details, please see source codes.


Running the Sample

Please follow the steps below to run a X-Lite with this sample:

  1. Copy echo directory (or echo.sar) into $INSTALL_DIR/apps directory. Start Prism Application Server if you haven't.

  2. Startup and Configure your X-Lite (or other compatible SIP Clients).

    • Open X-Lite 3.0 or later.
    • On the display board, write click and select "SIP Acount Settings...".


    • On the SIP Accounts panel, click Add....


    • Type the User Details and Domain Proxy as follow.
    • Then click OK.


  3. Register your X-Lite to "Echo Servlet" application.

    • Start SIP servlet container (Prism Application Server) as user guide.
    • Start X-Lite,and it will register to the container.


  4. Use your X-Lite to chat with "Echo Servlet" application.

    • Click the right arrow and right click on the Group List,select "Add Contact...".


    • On the "Add Contact" tab, select "Softphone" as the Type,type the name: echo, and then click OK.Don't forget to select"Show this contact's Availability".


    • Right click on the "echo" ,and select "Instant Message".



    • Type your message in the box at the bottom of the window, and click Send.Now,you are chatting with echo.
