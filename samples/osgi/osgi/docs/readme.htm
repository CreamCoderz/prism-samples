OSGi Sample

Overview

This sample application demonstrates how to write a application act as a OSGi bundle. It also demonstrates how to easily using the OSGi service.

The OSGi Specifications provide more details the OSGi technology.


Understanding the Sample

OSGiServlet is a simple HTTP servlet. It demonstrates how servlet and OSGi container can exchange information via ServletContext and BundleServlet. OSGiServlet invokes the ConfigurationAdmin service to get all configurations of OSGi Container by OSGi service management mechanism. When OSGiServlet receives HTTP GET or POST request, it builds a HTML page to display the list of the configurations.

For more details, please see source codes.


Running the Sample

The top level directories and files in this sample are:

docs/ the documentation about this sample
osgi.war the application archive file (WAR)
osgi/ the exploded application directory

Please follow the following steps to deploy, configure and run this sample:

  1. Copy osgi.war into $INSTALL_DIR/modules directory.

  2. Access the URL http://127.0.0.1:8080/osgi
